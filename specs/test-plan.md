# Anchor E2E Test Plan

last_updated: 2026-04-28

## 통계

- **P0**: 28개 (매일 CI — 핵심 기능 생존 지표)
- **P1**: 24개 (주 1회 — 기능 회귀 방지)
- **P2**: 15개 (수동 트리거 또는 특이사항 발생 시)
- **VR**: 10개 (visual regression, 매일 또는 주 1회)

---

## 자동화 범위 외 (사람 QA 필요)

- UX 평가 (직관성, 카피 적절성, 모바일 레이아웃)
- 인맥 추천 알고리즘 결과의 도메인 정확성
- 세무이력 리포트 차트 데이터 정합성
- 법인 그룹 분류(1그룹/2그룹) UI 추가 시 재검증
- 새 기능 첫 출시 수동 검증
