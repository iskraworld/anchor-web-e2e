# 2. 테스트 시나리오

## 2.1. 세무사/세무법인 시나리오

### 시나리오 A — 특정 공무원에 대한 공통 인맥 탐색 (세무법인 소속 전관 세무사) : [bagseongho@gaon.com](mailto:bagseongho@gaon.com)

**상황**

- 공무원 출신 세무사가 서울청 재산팀 김정희 조사관과의 공통 인맥 파악 필요

**서비스 이용 경험**

- 이 공무원과
    1. 내가 다이렉트로 아는가?
    2. 나와 가까운 인맥이 있는가?
    3. 우리 회사 사람과 가까운 인맥이 있는가?
    4. 이 사람에게 제일 영향력이 높은 사람은 누구인가?

**데모 흐름:**

1. 현직 공무원 탐색에서 서울청 >성실납세지원국 > 소득재산세과 > 재산팀 김정희 조사관을 검색한다.
2. 김정희 조사관의 프로필을 확인하고, 나와의 공통 인맥 그래프를 조회한다.
    - 직접 관계는 없고, 2명의 공통 인맥이 확인된다. → 관계가 강하지는 않다.
3. 김정희의 인사권자까지 포함한 공통관계 그래프를 조회한다.
    - 김진범 팀장(상사)를 내가 직접 알 가능성이 있다. → 같은 대학 동문으로, 관계가 강하지는 않다.
    - 김정희 담당자, 김진범 팀장(상사), 최승일 과장(상사의 상사)과 5명의 공통 인맥이 확인된다. → 관계가 강하지는 않다.
4. 1그룹 전체 인원과 김정희+김정희의 인사권자 간 접점을 확인한다.
    - 김진범 팀장(상자)와 직접 관계인 5명의 세무사가 확인된다. → 관계가 강하지는 않다.
    - 김정희 담당자, 김진범 팀장(상사), 최승일 과장(상사의 상사)과 28명의 공통 인맥이 확인된다.
        - 박성훈 세무사와 동창(같은 대학, 같은 기수) 관계인 고영호 전관이 김진범의 동문(같은 대학)/옛 상사/동향 관계로 영향력이 가장 높다.
5. 혹시 모르니 우리 공통 인맥이 아니지만 김정희와 인사권자에게 영향력 있는 사람 명단을 살펴 본다.
    - 최승일 과장(상사의 상사)과 김정희에 대해서는 영향력이 높은 사람이 확인되지 않으나, 김진범 팀장(상사)에게 영향력 점수가 높은 사람은 여럿 확인된다.
    - 현직공무원 김상원의 프로필을 클릭하고 나와의 관계를 본다. → 마땅치 않으므로 박성훈 동료를 통해 김진범에게 접근한다.

### 시나리오 B — 같은 팀 공무원 출신 세무사의 인맥 조회 (세무법인 소속 일반 세무사) : taxhan@theanchor.best

**상황:** 비공무원 출신 세무사가 서울청 동대문세무서 조사과 조사팀 김경국 조사관과의 공통 인맥 파악 필요

서비스 이용 경험

- 이 공무원과
    1. 우리 회사 사람과 가까운 인맥이 있는가?
    2. 이 사람에게 제일 영향력이 높은 사람은 누구인가?

**데모 흐름:**

1. 현직 공무원 탐색에서 서울청 > 동대문 세무서 > 조사과 > 조사팀 김경국 조사관을 검색한다.
2. 본인 기준으로는 공통관계 조회가 안되므로 회사 전체와의 관계 그래프를 조회한다.
    - 양쪽에 관계 선이 가장 진한 김수희 현직 공무원을 선택한다.
        - 박성호 선배와 입사 동기/동창이다.
3. 박성호 선배를 통해 김수희 현직공무원에게 연락한다.

### 시나리오 C — 프로필 등록 후 납세자 노출 확인 (세무법인 소속 일반 세무사) : taxhan@theanchor.best

**진입 동기:** "납세자가 검색했을 때 내가 보이는지 확인하고 싶다."

1. 세무사 검색화면에서 가온세무법인을 검색한다.
2. 다른 사람들은 프로필이 조회되는데 자기만 빈약한 것을 확인한다.
3. 세무이력 관리 화면에 접속해 자기 이력을 등록한다.

3. 2그룹 파트너에게 개선을 요청한다.
