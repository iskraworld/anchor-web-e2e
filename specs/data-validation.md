## ⚠️ 주의: 시나리오 조정 필요 항목

### TM-C-1: 한지희 토글 상태 불일치

- **문제**: 시나리오 전제는 한지희 토글 OFF, 실제 staging은 ON 상태
- **조치**: TM-C-1은 staging 데이터 원복 전까지 **CONDITIONAL** 처리
- **Anchor 팀 요청**: 한지희(taxhan@theanchor.best) 세무사 프로필 노출 토글 **OFF** 상태로 원복 필요
- **UUID**: `2149aa48-d0af-4437-b978-f02644c9baad`

### TM-F-3: 박성호 소속 기관 불일치

- **문제**: 시나리오에 "가온세무법인 광주 서구"로 명시, 실제 프로필엔 "금융분석원" 표시
- **조치**: 소속 법인 정확 매치 대신 "소속 기관 정보가 표시된다" (비어있지 않음) 유연 검증으로 수정
- **Anchor 팀 요청**: 박성호 소속 데이터 확인 (가온세무법인 vs 금융분석원)

### TM-F-3: 문의 버튼 검증 제거

- **문제**: 세무사 프로필에 전용 문의 버튼 없음 (footer의 외부 링크만 존재)
- **조치**: "문의 또는 연락 버튼이 존재한다" 검증 항목 제거, 프로필 정보 렌더링 확인으로 대체

### TM-D-2, TM-D-3: 그룹 분류 없음

- **문제**: 법인 리포트(`/tax-history-report/me`)에 1그룹/2그룹 분류 UI 없음
- **조치**: **BLOCKED** — Anchor 팀이 그룹 분류 기능 추가 후 재검토
- **Anchor 팀 요청**: 가온세무법인 1그룹/2그룹 분류 기능 릴리즈 일정 및 UI 확인
