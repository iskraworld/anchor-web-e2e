# Phase 2: AI 자율 탐색 (2시간)

## 목표

옵션 B 범위로 anchor를 자율 탐색해서 Full QA 시나리오 생성.

**옵션 B 범위**:
- 모든 페이지 200 OK + 콘솔 에러 체크
- 헤더·사이드바·푸터 모든 링크
- 권한별 접근 차이 (5개 계정)
- 모든 폼의 valid/invalid validation
- 검색·필터·정렬 조합
- Visual regression baseline (모든 페이지 스크린샷)

**옵션 B 제외**:
- 모든 인터랙션 조합 (폭발적 증가)
- 깊은 edge cases
- 도메인 깊은 검증 (이건 사람 QA)
